# Programming Constructs

This module builds the mental model you need before deep DSA and large-scale systems. It covers representation, cost analysis, basic data organization, and clean program structure.

Source curriculum topic list:
[Scaler Academy Curriculum](https://www.scaler.com/academy/?no_redirect=true#curriculum-features)

## Topics in this module

- [Number Systems](./programming-constructs/01-number-systems.md)
- [Bit Manipulation](./programming-constructs/02-bit-manipulation.md)
- [Time Complexity](./programming-constructs/03-time-complexity.md)
- [Arrays](./programming-constructs/04-arrays.md)
- [Sorting](./programming-constructs/05-sorting.md)
- [Hashing](./programming-constructs/06-hashing.md)
- [Recursion](./programming-constructs/07-recursion.md)
- [OOP](./programming-constructs/08-oops.md)

## Outcomes

By the end of this module, you should be able to:

- understand how data is represented internally
- estimate time and space cost before coding
- use arrays, hashing, and recursion comfortably
- write small object-oriented designs cleanly

## Suggested approach

1. Read the theory page.
2. Run the example code yourself.
3. Re-implement the code from memory.
4. Solve 5 to 10 problems per topic.
5. Revise mistakes before moving on.
