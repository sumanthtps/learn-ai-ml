# Scaler handbook overview

This folder contains a topic-wise handbook built around the Scaler Academy curriculum topics and extended with a practical Generative AI track.

Base curriculum source:
[Scaler Academy Curriculum](https://www.scaler.com/academy/?no_redirect=true#curriculum-features)

This is a synthesized study guide, not Scaler's official teaching material. The curriculum topics come from the page above, and the notes expand them into full learning material with explanations, examples, code, project ideas, and project solutions.

## What is inside this folder

- [Programming Constructs](./02-programming-constructs.md)
- [Data Structures and Algorithms](./03-data-structures-and-algorithms.md)
- [System Design](./04-system-design.md)
- [Real-world Projects](./05-real-world-projects.md)
- [Generative AI](./06-generative-ai.md)

## Recommended study order

1. Programming Constructs
2. Data Structures and Algorithms
3. System Design
4. Real-world Projects
5. Generative AI

## How to use each topic page

Each topic page is organized around:

- concept and intuition
- where it is used
- worked examples
- code examples
- common mistakes
- interview and project relevance

## Core expectations

By the end of this folder, you should be able to:

- explain and implement core CS foundations
- solve interview-grade DSA questions with confidence
- reason about schema, scaling, caching, and architecture
- ship backend, frontend, full-stack, and GenAI projects
- understand the building blocks behind LLM apps and AI systems

## Suggested weekly cadence

- `3 days`: theory and notes
- `2 days`: coding practice
- `1 day`: project work
- `1 day`: revision and retrospective
