# System Design

This module moves from writing correct code to designing dependable systems under real-world constraints like traffic, storage growth, failures, and team scale.

Source curriculum topic list:
[Scaler Academy Curriculum](https://www.scaler.com/academy/?no_redirect=true#curriculum-features)

## Outcomes

- state system requirements before choosing tools
- model schemas around access patterns
- talk clearly about cache, queues, scale, and tradeoffs
- connect low-level design to high-level system behavior
